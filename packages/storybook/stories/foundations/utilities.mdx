# Utilities

Utility classes are a small opt-in layer for app composition. They map directly to design-system tokens and live in the `ds.utilities` cascade layer, so consumer app CSS can still override them without `!important`.

They are intentionally limited to spacing, layout, and typography. Use components and component parts for colors, borders, shadows, and component-like visual styling.

## Imports

Import one category when you only need that part of the utility layer:

```ts
import '@jsekulowicz/ds-components/css/utilities/spacing.css';
import '@jsekulowicz/ds-components/css/utilities/layout.css';
import '@jsekulowicz/ds-components/css/utilities/typography.css';
```

Or import the combined bundle:

```ts
import '@jsekulowicz/ds-components/css/utilities.css';
```

CSS utility files are not tree-shaken at the individual class level. The package exposes split CSS entry points so apps can keep the bundle small by importing only the categories they use. The CSS exports are marked as side effects so bundlers preserve explicit imports.

## Spacing

Spacing utilities use `--ds-space-*` tokens. The supported scale is `0` through `6`.

| Class | Property |
|---|---|
| `.ds-p-0` ... `.ds-p-6` | `padding` |
| `.ds-px-0` ... `.ds-px-6` | `padding-inline` |
| `.ds-py-0` ... `.ds-py-6` | `padding-block` |
| `.ds-pt-0` ... `.ds-pt-6` | `padding-block-start` |
| `.ds-pr-0` ... `.ds-pr-6` | `padding-inline-end` |
| `.ds-pb-0` ... `.ds-pb-6` | `padding-block-end` |
| `.ds-pl-0` ... `.ds-pl-6` | `padding-inline-start` |
| `.ds-m-0` ... `.ds-m-6` | `margin` |
| `.ds-mx-0` ... `.ds-mx-6` | `margin-inline` |
| `.ds-my-0` ... `.ds-my-6` | `margin-block` |
| `.ds-mt-0` ... `.ds-mt-6` | `margin-block-start` |
| `.ds-mr-0` ... `.ds-mr-6` | `margin-inline-end` |
| `.ds-mb-0` ... `.ds-mb-6` | `margin-block-end` |
| `.ds-ml-0` ... `.ds-ml-6` | `margin-inline-start` |
| `.ds-gap-0` ... `.ds-gap-6` | `gap` |
| `.ds-gap-x-0` ... `.ds-gap-x-6` | `column-gap` |
| `.ds-gap-y-0` ... `.ds-gap-y-6` | `row-gap` |

Example:

```html
<div class="ds-d-flex ds-items-center ds-gap-2 ds-p-4">
  ...
</div>
```

| Class | Property |
