# Skeleton

Decorative placeholder for content that has not loaded yet. Compose multiple `ds-skeleton` instances to match the eventual layout instead of inventing one-off shimmer styles.

  <ds-tab-panel value="guidelines">
    <h3>Do</h3>
    <ul>
      <li>Match the rough size of the content that will replace the skeleton.</li>
      <li>Use `text` for copy, `rectangle` for media or panels, and `circle` for avatars.</li>
      <li>Keep skeletons decorative; put loading announcements on the surrounding region.</li>
    </ul>
    <h3>Don't</h3>
    <ul>
      <li>Use skeletons for errors or empty states.</li>
      <li>Mix several unrelated placeholder layouts in the same region.</li>
    </ul>
  </ds-tab-panel>
